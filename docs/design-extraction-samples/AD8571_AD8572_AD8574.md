## Suggested topologies

- `Capacitive Load Driver` (pages: 19): The datasheet explicitly recommends an RC snubber when driving capacitive loads.
- `Strain Gage Front End` (pages: 20): Bridge excitation, reference generation, and gain staging are all needed for strain-gage modules.
- `Thermocouple Front End` (pages: 21): Cold-junction compensation and gain staging are needed for thermocouple sensing.
- `Current Sense Front End` (pages: 21): A sense resistor and gain path are present in the extracted equations and component hints.

## L3 Templates

- `A1_non_inverting_gain` `Non-Inverting Gain Stage`: Use for sensor buffering or moderate gain where input impedance should stay high.
- `A1_inverting_gain` `Inverting Gain Stage`: Use when source impedance is controlled and polarity inversion is acceptable.
- Start here: `A1_cap_load_driver` `Capacitive Load Driver`: Use when the load is capacitive or the datasheet recommends an RC snubber for stability.
- `A1_strain_gage_frontend` `Strain Gage Front End`: Use for bridge sensors or weigh-scale front ends that need excitation and differential amplification.
- `A1_thermocouple_frontend` `Thermocouple Front End`: Use for low-level thermocouple signals that need cold-junction compensation and gain.
- `A1_current_sense_frontend` `Current Sense Front End`: Use when RSENSE-based monitoring is called out in the datasheet equations or examples.
