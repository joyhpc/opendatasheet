## First-pass checklist

- Add `CIN` for `input_decoupling` (`required`): Every module should start with local input decoupling near the device power pin.
- Add `REN` for `enable_bias` (`optional`): A bias resistor or logic source prevents ambiguous enable-state bring-up.
- Add `AUTO_INPUT_CAPACITOR` for `input_capacitor` (`datasheet_hint`): ADG707 ±0.2 ±1 nA max Suggested values: 5 pF.
- Add `AUTO_OUTPUT_CAPACITOR` for `output_capacitor` (`datasheet_hint`): Input High Voltage, VINH 2.4 V min Suggested values: 35 pF.
