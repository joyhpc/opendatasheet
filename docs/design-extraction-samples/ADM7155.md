## First-pass checklist

- Add `CIN` for `input_decoupling` (`required`): Every module should start with local input decoupling near the device power pin.
- Add `COUT` for `output_capacitor` (`required`): LDO stability usually depends on the output capacitor value and ESR window.
- Add `REN` for `enable_bias` (`optional`): A bias resistor or logic source prevents ambiguous enable-state bring-up.
- Add `AUTO_INPUT_CAPACITOR` for `input_capacitor` (`datasheet_hint`): 600 mA, Ultralow Noise, Suggested values: 10 µF.
- Add `AUTO_OUTPUT_CAPACITOR` for `output_capacitor` (`datasheet_hint`): 600 mA, Ultralow Noise, Suggested values: 10 µF.
- Add `AUTO_GAIN_RESISTOR` for `gain_resistor` (`datasheet_hint`): 600 mA, Ultralow Noise,
- Add `AUTO_FEEDBACK_DIVIDER` for `feedback_divider` (`datasheet_hint`): CIN COUT

## Key constraints

- `vin_abs_max`: Input Voltage (min=-0.3, max=7.0, unit=V)
- `vin_operating`: Input Voltage Range (min=2.3, max=5.5, unit=V)
- `vout_range`: Output Voltage Range (min=1.2, max=3.4, unit=V)
- `iout_max`: Maximum Load Current (max=600, unit=mA)
- `quiescent_current`: Quiescent Current (typ=4, unit=mA)
- `dropout_voltage`: Dropout Voltage (typ=120, unit=mV)
