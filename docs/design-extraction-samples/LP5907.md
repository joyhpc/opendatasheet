## First-pass checklist

- Add `CIN` for `input_decoupling` (`required`): Every module should start with local input decoupling near the device power pin.
- Add `COUT` for `output_capacitor` (`required`): LDO stability usually depends on the output capacitor value and ESR window.
- Add `REN` for `enable_bias` (`optional`): A bias resistor or logic source prevents ambiguous enable-state bring-up.
- Add `AUTO_INPUT_CAPACITOR` for `input_capacitor` (`datasheet_hint`): 5 Pin Configuration and Functions......................... 4 9 Power Supply Recommendations...................... 17
- Add `AUTO_OUTPUT_CAPACITOR` for `output_capacitor` (`datasheet_hint`): LP5907 Suggested values: 1 µF.
- Add `AUTO_GAIN_RESISTOR` for `gain_resistor` (`datasheet_hint`): 8 Application and Implementation

## Key constraints

- `vin_abs_max`: Input voltage (min=-0.3, max=6.0, unit=V)
- `vin_operating`: Input voltage (min=2.2, max=5.5, unit=V)
- `iout_max`: Load current (min=0, max=250, unit=mA)
- `current_limit`: Short-circuit current limit (min=250, typ=500, unit=mA)
- `thermal_shutdown`: Thermal shutdown (typ=160, unit=°C)
- `quiescent_current`: Quiescent current (typ=12, max=25, unit=μA)
- `dropout_voltage`: Dropout voltage (typ=120, max=200, unit=mV)
- `output_capacitance`: Output capacitance (min=0.7, typ=1, max=10, unit=μF)
- `output_cap_esr`: Output/Input capacitance ESR (min=5, max=500, unit=mΩ)
