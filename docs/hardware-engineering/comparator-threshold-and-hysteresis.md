# 比较器阈值与迟滞设计

## 适用场景

适用于欠压过压检测、电流门限、零交越检测、按键整形和窗口比较。

## 关键规则

- 比较器不是低速运放，输出级、传播延迟、输入共模和恢复行为都不同。
- 没有迟滞的阈值检测在噪声边缘几乎必然抖动。
- 迟滞大小应与输入噪声、纹波和容差预算匹配，不是越大越好。
- 开漏输出的上拉、默认态和逻辑域必须提前定义。

## 评审清单

- 输入共模范围是否覆盖整个比较区间。
- 正反馈网络是否形成了可计算的迟滞窗口。
- 参考源是否稳定且不过度受输入分压负载影响。
- 输出翻转后的逻辑域、电平和延迟是否符合系统需求。
- 上电和慢爬升条件下是否会误翻转。

## 常见失误

- 用通用运放替代比较器做高速阈值检测。
- 加了迟滞但没有重新核算真实门限。
- 开漏输出缺少上拉或上拉到错误电压域。
