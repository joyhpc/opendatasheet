# 以太网 PHY Bring-up 清单

## 适用场景

适用于 10/100/1000BASE-T PHY、MAC+PHY 板级接口和 RJ45 外部口 bring-up。

## 关键规则

- 先确认时钟、复位、strap、供电和 MDIO 可访问，再看链路训练。
- 磁性器件、共模、电源滤波和中心抽头供电必须遵循 PHY 参考设计。
- RJ45 口的 ESD、Bob-Smith、机壳地与数字地策略需要一致。
- RGMII/GMII/RMII 板内接口和线侧差分接口是两套不同问题，不要混着看。

## 评审清单

- PHY 主时钟、25 MHz/50 MHz/125 MHz 源是否稳定且匹配模式。
- 复位时序和 strap 上拉下拉是否满足上电采样窗口。
- MDIO 地址和寄存器访问是否打通。
- 线侧变压器、中心抽头、电源滤波和终端是否按应用笔记实现。
- RGMII 延迟策略是在 PHY 内部加还是板上加，是否前后一致。

## 常见失误

- strap 电阻值正确但上电窗口错误，导致模式读错。
- RJ45 屏蔽壳接地策略模糊，既想隔离又想直连。
- 只关注 PHY 供电电压，不看模拟 rail 噪声和时钟品质。
