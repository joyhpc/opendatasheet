# 器件降额设计指南

## 适用场景

适用于工业、车载、长寿命设备和对可靠性有明确要求的电子产品。

## 关键规则

- 降额不是保守主义，而是把器件从“能工作”拉回“长期可靠工作”。
- 电压、电流、功耗、结温、纹波电流和浪涌能力都应单独降额。
- 降额策略必须与应用场景匹配，不能用消费电子经验套工业和车载。
- 高温、密闭箱体、脉冲负载、插拔应力下的降额应更严格。

## 评审清单

- 电容、电感、MOSFET、二极管、保险丝是否有明确降额标准。
- 电解、电容、LED、磁性件是否考虑温度和寿命模型。
- TVS、保险丝和功率件是否按脉冲而非平均功率评估。
- 降额是否体现在选型文档和 BOM 备注，而不是口头约定。
- 替代料是否重新做过降额验证。

## 常见失误

- 只给电压做降额，不给热和脉冲做降额。
- 用典型环境温度计算结温。
- 换了更便宜替代料却沿用原设计的可靠性假设。
