# 可制造性 DFM 快速检查

## 适用场景

适用于：

- 打样前最后一轮硬件评审
- 量产导入前的工艺风险筛查
- 封装替换或板厂切换时的复核
- 高密度 BGA / QFN / 细间距连接器板卡

## 不适用场景

不适用于只做电气 ERC/DRC 的场景。本文关注的是“板子电气上成立，但工艺上不稳”的问题。

## 典型失效症状

DFM 没过关，最常见表现不是“设计错误”，而是：

- 样板能焊出来，但良率差
- 热焊盘器件空洞大、偏移大、返修难
- 某些封装在不同 SMT 厂表现完全不同
- ICT/FCT 工装难以下针
- 小批量能做，大批量问题集中爆发

## 先看什么

先别看功能模块，先看四个工艺高风险面：

1. 细间距封装
2. 热焊盘与过孔
3. 测试可达性
4. 拼板、定位、返修空间

## 必查顺序

### 1. 封装与焊盘

- BGA、QFN、细间距连接器的焊盘与阻焊桥是否匹配板厂能力。
- 逃线、过孔、阻焊开窗是否会压缩焊接窗口。
- 替代料封装若略有差异，是否重新过工艺检查。

### 2. 热焊盘与过孔

- 热焊盘器件是否考虑钢网开窗和空洞控制。
- 焊盘中或焊盘附近过孔是否需要塞孔、盖孔或其它控锡策略。
- 是否存在“电气可行，焊接极不稳定”的 exposed pad 设计。

### 3. 测试与返修

- 关键网络是否有量产测试可达性。
- 高风险器件是否预留返修工具空间。
- 测试点、Fiducial、定位孔是否满足生产线要求。

### 4. 拼板与装配

- 器件高度、禁布区、拼板边、连接器朝向是否适配装配工艺。
- 大铜皮、小焊盘、热容量不均是否会造成焊接偏差。
- SMT 厂和 PCB 厂的能力边界是否已经确认。

## 硬规则

- DFM 不是板厂最后兜底，原理图和封装阶段就必须介入。
- 热焊盘器件必须同步审查焊盘、钢网、过孔和返修策略。
- 高密度板卡没有测试可达性，就不是成熟量产设计。
- 若板厂/SMT 厂能力不明确，不应假设“他们总能做出来”。
- 任何封装替换都要重新过 DFM，而不是只过电气检查。

## 常见失误

- 只看 ERC/DRC，不问工艺窗口。
- 热焊盘过孔没控锡，结果空洞和偏移严重。
- 返修空间和测试可达性到样板后才被提出。
- 认为小批量能焊出来，就代表量产可控。
