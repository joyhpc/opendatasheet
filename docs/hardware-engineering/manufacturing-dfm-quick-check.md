# 可制造性 DFM 快速检查

## 适用场景

适用于打样前评审、量产导入、封装替换和高密度板卡的工艺风险筛查。

## 关键规则

- DFM 不是 PCB 厂最后帮你兜底的事情，原理图和封装阶段就已经决定了大半风险。
- BGA、QFN、细间距连接器、热焊盘器件和大电流铜皮都应提前做工艺审查。
- 测试可达性、拼板、定位孔、禁布区和器件高度限制同样属于 DFM。
- 工艺风险最好在样板前就和 PCB/SMT 厂闭环。

## 评审清单

- 细间距封装的焊盘、阻焊桥和过孔策略是否适合板厂能力。
- 热焊盘器件是否考虑钢网开窗和空洞控制。
- 大铜皮与小焊盘连接是否需要热焊盘或泪滴优化。
- 关键器件是否预留返修空间和可视检查条件。
- 拼板、Fiducial、测试点、定位孔是否满足生产线要求。

## 常见失误

- 只看电气规则，不问板厂和 SMT 厂是否做得稳。
- 热焊盘过孔没控锡，结果焊接空洞和偏移严重。
- 高器件密度板没有考虑返修工具可达性。
