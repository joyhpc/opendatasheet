# FPGA 收发器参考时钟

## 适用场景

适用于 PCIe、Ethernet、JESD、Aurora、GMSL/FPD-Link 桥接和高速收发器系统。

## 关键规则

- 参考时钟质量要按链路协议和 transceiver 手册定义，不能只按频率正确判断。
- 差分阻抗、AC 耦合位置、源端/接收端布线长度和回流连续性必须统一审视。
- 参考时钟电源通常应视为敏感模拟电源。
- 时钟分配器和连接器会引入附加抖动与 skew，必须计入预算。

## 评审清单

- 参考时钟的制式、摆幅、共模和抖动是否满足器件要求。
- AC 耦合电容位置是否遵循官方板级建议。
- 参考时钟与数据信号是否分层布线并避开高噪声区域。
- clock source、fanout、connector、FPGA 输入之间是否有完整 jitter budget。
- 双参考时钟或冗余时钟方案是否有明确切换策略。

## 常见失误

- 用通用 XO 直接驱动高要求 transceiver 参考时钟。
- AC 耦合电容放在线路中间，制造额外 stub。
- 参考时钟电源共用开关噪声严重的 rail。
