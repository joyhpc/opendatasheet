# 散热过孔与铺铜扩热

## 适用场景

适用于功率器件、LDO、DC/DC、LED 驱动、功率放大器和高功耗数字器件。

## 关键规则

- 散热铜皮和过孔要服务真实热路径，不能只追求“看起来很多过孔”。
- 裸露焊盘器件应优先把热引到连续大面积地或电源铜皮。
- 热过孔阵列既影响热阻，也影响焊接工艺和空洞风险。
- 强制风冷和自然对流下的最优铜皮策略不完全一样。

## 评审清单

- 功耗最大的器件是否有明确的主散热路径。
- Exposed pad 是否接到足够面积的铜和热过孔阵列。
- 热过孔是否需要塞孔、盖孔或工艺控制以避免焊锡流失。
- 相邻热源是否互相堆叠加热。
- 热扩散铜皮是否与高噪声电流回路冲突。

## 常见失误

- 只增加铜皮面积，却没有把热从封装热焊盘有效导出。
- 热过孔数量很多但都离热源太远。
- 为了散热把敏感模拟地和功率地硬连成大噪声面。
