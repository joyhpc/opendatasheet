# ADC 基准与输入驱动

## 适用场景

适用于：

- SAR ADC 前端
- Delta-Sigma ADC 模拟输入链路
- 精密采样系统
- 传感器到 ADC 的缓冲、滤波与参考设计

## 不适用场景

不适用于只关心 ADC 数字接口时序的场景。本文关注的是模拟链路能否把 datasheet 上的 ENOB 变成板上的真实结果。

## 典型失效症状

ADC 前端没处理好，常见表现是：

- 静态看起来正常，但动态采样误差大
- 低频精度尚可，高速采样或多通道切换时明显恶化
- 某些输入范围线性度差
- 噪声底抬高，码值抖动异常
- 更换参考源或采样率后结果突然变化

## 先看什么

先回答三个问题：

1. 参考源是否稳定且独立。
2. 驱动器是否能在采样窗口内建立。
3. 输入 RC 网络是在帮你，还是在偷偷破坏稳定性。

## 必查顺序

### 1. 参考源

- 参考源输出噪声、温漂、电流能力是否满足 ADC 动态需求。
- 参考去耦是否按 datasheet 推荐就近放置。
- 参考是否被别的数字或开关负载共享。

### 2. 输入驱动

- 源阻抗是否适合 ADC 输入结构。
- 前级运放或缓冲是否能驱动采样电容充放电。
- 共模范围、全量程和摆幅是否完全覆盖目标信号。

### 3. RC 与稳定性

- 输入 RC 是否既满足抗混叠，又不破坏驱动建立。
- SAR ADC 前的串阻是否过大，导致采样电容来不及充满。
- Delta-Sigma 前端滤波是否与数字滤波和输入带宽目标一致。

### 4. 接地与回流

- AGND、REFGND、DGND 是否按器件手册连接。
- 数字回流是否穿过参考和模拟输入附近。
- 参考源和驱动器是否避开高 di/dt 区域。

## 硬规则

- 高源阻抗信号不能默认直接驱动 SAR ADC。
- 参考源不能顺手共享给会注入动态电流的其它负载。
- 输入 RC 值必须和 ADC 输入结构一起算，不能只按经验取。
- ADC 前端 review 时，必须把参考、驱动、RC、接地当成一个整体看。
- 没有建立时间预算，就不要相信“理论分辨率”。

## 常见失误

- 直接让桥式、热敏、电位器等高源阻抗信号直驱 SAR ADC。
- 参考源共享给 DAC、逻辑或其它模拟块。
- 只看 ADC datasheet 的 SNR/ENOB，不看前端是否满足测试条件。
