# ADC 基准与输入驱动

## 适用场景

适用于 SAR、Delta-Sigma、高速 ADC 和精密采样前端。

## 关键规则

- ADC 的核心不只是分辨率，前端驱动能力、参考稳定性和采样瞬态同样决定有效精度。
- 参考电压源必须低噪声、低漂移并有正确去耦，不能挂任意数字负载。
- SAR ADC 常见的采样电容会在采样瞬间向前级拉电荷，需要驱动器和 RC 网络共同设计。
- 输入滤波器既服务抗混叠，也服务驱动稳定性。

## 评审清单

- 输入驱动器是否能在采样窗口内完成建立。
- 参考源输出电流和去耦是否满足 ADC 动态需求。
- 模拟输入 RC 是否与采样保持电容和驱动器稳定性兼容。
- AGND、REFGND、DGND 连接是否按器件建议实现。
- 全量程、共模、输入保护和过压条件是否覆盖。

## 常见失误

- 直接让高源阻抗传感器驱动 SAR ADC。
- 参考源共享给其他开关负载。
- 只看 ADC datasheet 的 ENOB，不看前端建立时间和驱动拓扑。
