# Bring-up 实验室仪器配置

## 适用场景

适用于新板首电、接口调试、电源稳定性验证和问题复现环境搭建。

## 关键规则

- Bring-up 的效率取决于测量链是否可靠，而不只是板子设计本身。
- 第一轮最有价值的仪器通常是可限流电源、示波器、逻辑分析、万用表和稳定串口/JTAG 工具。
- 对高速和电源问题，探头接法、地线长度和带宽设置常常比示波器型号更重要。
- 实验室供电、线缆、转接板和工装必须标准化，否则问题不可重复。

## 评审清单

- 是否准备了分 rail 限流和电流监视能力。
- 是否有足够通道数同时观察电源、复位、时钟和关键接口。
- 探头、同轴、差分探头、电流探头是否适配目标问题。
- 串口、JTAG、USB、以太网等通道是否有已知可靠的工装。
- 实验记录模板、波形保存和日志命名是否一致。

## 常见失误

- 用长地线探头测开关节点，得到大量假振铃。
- 只有板子没有工装，每次接线都不一样。
- 电源限流设置过高，首电就烧掉故障点。
