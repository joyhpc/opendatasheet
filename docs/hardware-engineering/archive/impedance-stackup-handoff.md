# 阻抗与叠层交付要求

## 适用场景

适用于高速板卡、量产交付、外部 PCB 厂协同和 SI 前置评审。

## 关键规则

- 叠层不是 layout 结束后再找板厂确认的工艺文件，而是接口设计输入。
- 必须明确目标阻抗、层间介质、铜厚、参考层和可接受公差。
- 同一网络在不同层的传播速度不同，长度规则必须考虑层别。
- 若设计依赖回钻、背钻、埋盲孔或特殊树脂，需在立项阶段就锁定。

## 评审清单

- 每类关键网络的目标阻抗和所用层是否列表化。
- Stackup 是否区分高速层、敏感模拟层、电源层和地层职责。
- 是否定义了板厂阻抗 coupon 和测试要求。
- 长度匹配规则是否已按层间速度差进行修正。
- 连接器、FPC、同轴过渡是否纳入同一阻抗链路考虑。

## 常见失误

- 原理图完成后才发现板厂常用叠层无法满足目标阻抗。
- 高速网络被路由到未定义阻抗的临时层。
- 工艺能力和 SI 规则脱节，导致量产前大改 stackup。
