# 时钟源与晶振选择

## 适用场景

适用于 MCU、FPGA、PHY、SerDes、ADC、音频和无线前端的时钟设计。

## 关键规则

- 先明确需求是频率精度、相位噪声、抖动、启动时间还是低功耗。
- 晶体、MEMS、XO、TCXO、OCXO 的取舍取决于系统预算，不是品牌偏好。
- 时钟源供电噪声和布局常常比标称精度更影响最终系统表现。
- 不同负载对时钟的敏感指标不同，PHY 看 RMS jitter，RTC 看 ppm，采样系统看相位噪声。

## 评审清单

- 负载电容、驱动能力和晶体 ESR 是否与主控振荡器兼容。
- 时钟器件供电是否单独去耦并避开大电流回路。
- 输出制式是 CMOS、LVDS、LVPECL 还是差分 HCSL，终端是否匹配。
- 多个时钟域之间是否需要缓冲、分配或清洗。
- 启动时间、温漂和老化是否覆盖系统使用场景。

## 常见失误

- 只按频率选晶体，不核对负载电容和 ESR。
- 时钟旁边布了开关电源或大电流 via 墙。
- 用普通 XO 驱动严苛抖动预算的高速 SerDes。
