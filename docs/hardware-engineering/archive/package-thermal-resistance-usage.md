# 封装热阻的正确用法

## 适用场景

适用于器件选型、热预算、功耗评估和封装比较。

## 关键规则

- `θJA` 不是一个与应用无关的绝对常数，它依赖测试板、铜面积、风速和安装条件。
- 比较两个器件热性能时，要确认测试条件是否可比。
- `θJC` 更适合评估壳体到散热器路径，`θJA` 更适合粗略板级估算。
- 结温估算应使用最坏功耗和最坏环境，不要用典型值安慰自己。

## 评审清单

- 数据手册热阻表对应的测试条件是否被理解。
- 结温预算是否采用 `Tj = Ta + P * θ` 的保守估算。
- 高功耗场景下是否考虑了 airflow、铜皮和散热器的影响。
- 封装比较是否混淆了裸片能力和板级能力。
- 长期可靠性温度裕量是否足够。

## 常见失误

- 直接拿 `θJA` 做跨封装绝对排名。
- 只算室温，不算高温环境或密闭箱体。
- 忽略功率循环和热点集中，认为平均功耗低就一定安全。
