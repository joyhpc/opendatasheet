# 运放前端评审要点

## 适用场景

适用于传感器调理、基准缓冲、低速放大、滤波、差分转单端和通用模拟前端。

## 关键规则

- 先确认输入共模范围、输出摆幅、带宽、噪声、偏置电流和负载能力是否匹配。
- 单电源系统要特别检查输入是否触到轨、输出是否真能到轨。
- 反馈网络阻值影响噪声、偏置误差、功耗和带宽，不能随手取值。
- 有电容性负载时必须评估稳定性，不要默认运放都能直接驱动。

## 评审清单

- 输入源阻抗是否与运放输入偏置和噪声要求匹配。
- 反馈和输入电阻是否在合理数量级，不致放大漏电和噪声。
- 输出负载、线缆、电容或 ADC 采样电容是否会破坏相位裕量。
- 电源去耦、参考地和输入保护是否完整。
- 关键节点是否预留测试手段验证增益和噪声。

## 常见失误

- 用高速运放做低频高精度，却忽略偏置和漂移。
- 单电源下输入接近地或上电轨，结果线性区严重受限。
- 后级是采样 ADC 或长线缆，前级却未做稳定性补偿。
