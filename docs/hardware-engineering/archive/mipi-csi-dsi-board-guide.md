# MIPI CSI-2 / DSI 板级设计指南

## 适用场景

适用于摄像头、显示模组、桥接芯片和 SoC 间的短距高速差分互连。

## 关键规则

- MIPI 不是普通 LVDS，供电域、lane 数、时钟 lane、终端和上电时序都要按器件定义。
- 优先保证 lane 对内长度匹配、参考平面连续和连接器 pinout 合理。
- 连接器、FPC、板间转接对损耗和串扰影响很大，不能只盯 PCB 本体。
- 模组供电、复位、待机、I2C 控制和 MIPI lane 必须一起评审。

## 评审清单

- lane 数、极性、时钟 lane 方向是否与 SoC 和模组定义一致。
- 模组电源和 IO 电平是否满足上电时序要求。
- 差分对是否避免跨分割、换层过多和长 stub。
- FPC/BTB 连接器的阻抗和 pin 排布是否为高速对优化。
- MIPI 旁路的控制总线和 GPIO 是否具备稳定默认态。

## 常见失误

- 只对齐 lane 长度，却让参考平面多次中断。
- 摄像头 AVDD/DOVDD/DVDD 时序不满足，链路异常却误判为 SI 问题。
- FPC pinout 没为差分对和地回流做分组规划。
