# 高速连接器与逃线设计

## 适用场景

适用于板间高速连接器、Mezzanine、FMC、BTB、高速背板和 SoM-Carryer 接口。

## 关键规则

- 连接器不只是机械接口，它是信号通道的一部分。
- Pinout 设计应优先服务差分对、参考地、供电分组和回流路径。
- 逃线方式、过孔场和层切换会决定插损、反射和串扰。
- 对高速连接器，早期 pinout 优化通常比后期走线微调更有效。

## 评审清单

- 差分对两侧是否有足够地针或参考返回。
- 高速 lane 是否被分配到连接器中电气表现更好的区域。
- 逃线过孔和层切换数量是否被控制。
- 连接器额定速率与目标协议是否匹配。
- 供电针和高速针的相对位置是否会引入噪声耦合。

## 常见失误

- 先按机械方便定义 pinout，再要求 layout “想办法”实现高速。
- 差分对跨 connector pin 排列不对称，回流中断严重。
- 连接器数据手册只看针数和尺寸，不看 S 参数或速率等级。
