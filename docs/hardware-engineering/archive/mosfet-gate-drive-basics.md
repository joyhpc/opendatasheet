# MOSFET 栅极驱动基础检查

## 适用场景

适用于负载开关、同步整流、半桥、全桥、电机驱动和高边/低边开关设计。

## 关键规则

- 栅极驱动问题本质上是电荷、速度、损耗和 dv/dt 抗扰性的平衡。
- 不要只看阈值电压 `Vth`，实际导通能力应看目标栅压下的 `Rds(on)`。
- 高边 N-MOS 通常需要 bootstrap 或电荷泵，不能默认逻辑电平直接驱动。
- 栅极串阻、下拉、米勒钳位和死区时间都属于系统稳定性设计，不是“可有可无”。

## 评审清单

- 驱动器输出电流是否足够在目标频率下充放栅电荷。
- 栅极最大电压和负压瞬态是否被限制在器件额定内。
- 半桥上下管是否有可靠死区控制，避免直通。
- 栅极回路是否短，驱动地是否与功率地策略明确。
- dv/dt 触发、米勒效应和误导通风险是否被评估。

## 常见失误

- 用 MCU IO 直接驱动大 Qg MOSFET。
- 只按 `Vgs(th)` 认定 MOSFET “逻辑电平可驱动”。
- 栅极电阻随意填一个通用值，未看开关损耗和振铃。
