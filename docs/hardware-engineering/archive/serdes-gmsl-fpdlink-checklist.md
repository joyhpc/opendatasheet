# SerDes / GMSL / FPD-Link 评审清单

## 适用场景

适用于摄像头远距离链路、显示链路、同轴/双绞线串化器与解串器设计。

## 关键规则

- SerDes 设计同时包含高速链路、电源噪声、参考时钟、PoC 供电和控制通道。
- 不要把它简化成“只是一对高速差分线”。
- 同轴与 STP 的共模、ESD、PoC 注入、连接器和屏蔽策略完全不同。
- 参考设计中的 AC 耦合、EQ、反向通道配置和 strap 不应随意删改。

## 评审清单

- Serializer / Deserializer 的主供电、模拟供电、PLL 供电是否隔离审视。
- 参考时钟源、抖动和布线是否满足链路要求。
- PoC 网络的电感、电容、TVS 和滤波拓扑是否按器件推荐实现。
- 反向控制通道、GPIO、I2C pass-through 的电平和默认态是否明确。
- 连接器屏蔽、机壳地和线缆地策略是否与系统 EMI 策略一致。

## 常见失误

- 只关注链路通断，不关注 PoC 注入对信号完整性的影响。
- 参考时钟放在噪声电源或回流割裂区域。
- 忽略同轴外壳和机壳地关系，导致 ESD/EMI 风险。
