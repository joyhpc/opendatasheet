# DAC 输出缓冲设计

## 适用场景

适用于精密设定、电压偏置、波形输出、可编程阈值和闭环控制系统。

## 关键规则

- 先明确 DAC 是驱动静态高阻节点，还是驱动动态电容负载、远端线缆或采样网络。
- 缓冲运放必须同时满足稳定性、失调、噪声、摆幅和建立时间。
- 某些 DAC 输出需要外部参考缓冲、输出缓冲或低通滤波，不能裸接系统负载。
- 负载变化会直接影响 DAC 有效精度和毛刺表现。

## 评审清单

- DAC 输出是否需要缓冲运放隔离后级负载。
- 输出滤波是否会影响环路或更新速率。
- 参考源和 DAC 供电是否有独立去耦。
- 负载电容、采样负载或长线是否会导致缓冲不稳定。
- 电源上电、代码上电默认值和 reset 行为是否满足系统安全要求。

## 常见失误

- DAC 直接驱动长线或大电容负载。
- 缓冲运放只看精度，不看输出驱动和稳定性。
- 参考源布局靠近数字噪声节点，导致输出底噪恶化。
