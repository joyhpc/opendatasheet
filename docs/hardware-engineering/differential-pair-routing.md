# 差分对走线原则

## 适用场景

适用于 USB、PCIe、Ethernet、LVDS、MIPI、SerDes 和高速时钟差分线。

## 关键规则

- 差分设计的目标不是“永远两线完全并行”，而是控制差分阻抗、共模耦合和时序偏斜。
- 对内匹配、对间间距、参考平面连续和过孔处理优先于肉眼上的“绝对对称”。
- 差分对之间的串扰距离必须按层结构和速率定义，不要套一个固定经验值。
- 不同协议的目标阻抗、插损和 skew 预算不同，不能通用化。

## 评审清单

- 差分阻抗目标值是否来自协议或器件手册。
- 对内 skew 是否在接收端容限内。
- 是否跨过参考平面分割、连接器断层或大开窗。
- AC 耦合、电阻网络、ESD 器件是否保持对称放置。
- 差分对与其他 aggressor 之间是否保留足够间距。

## 常见失误

- 为了补长度做很多蛇形，反而引入额外串扰和损耗。
- 差分对一侧多次换层，另一侧基本不换层。
- 在差分线上随意挂测试点和 stub。
