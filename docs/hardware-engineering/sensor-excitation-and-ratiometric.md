# 传感器激励与比率测量

## 适用场景

适用于桥式传感器、应变、压力、称重、RTD 和电阻式位置传感器。

## 关键规则

- 激励源的稳定性、噪声和温漂会直接进入测量误差链。
- 当传感器输出与激励成比例时，优先考虑 ratiometric 架构。
- 激励与采样参考使用同一基准链时，可抵消一部分供电漂移误差。
- 长线传感器要同时评估激励压降、保护和 EMI。

## 评审清单

- 激励源是否满足精度、温漂和驱动能力要求。
- ADC 参考是否与激励形成合理的 ratiometric 关系。
- 传感器开路、短路、线缆反接是否可检测。
- 屏蔽、滤波、输入保护和共模范围是否完整。
- 激励开关控制是否会引入热漂和稳定时间问题。

## 常见失误

- 用不稳定数字电源直接激励高精度桥式传感器。
- 设计上宣称“比率测量”，但 ADC 参考与激励并不共源。
- 长线压降和保护网络造成激励失真却未被预算。
