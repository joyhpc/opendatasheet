# 模拟与数字混合接地

## 适用场景

适用于 ADC/DAC、传感器前端、音频、精密基准、MCU 混合信号板和电机控制。

## 关键规则

- 混合接地的核心是控制回流路径，不是盲目“分地再单点相连”。
- 现代 mixed-signal IC 往往要求在芯片附近统一参考地，再通过布局控制数字噪声回流。
- 地分割若切断了关键返回路径，会比“共地”更糟。
- 快速数字边沿、时钟、开关节点必须远离精密模拟输入和参考网络。

## 评审清单

- 高速数字回流是否穿过模拟区。
- ADC/DAC、参考源、运放周边是否有连续地参考和近端去耦。
- AGND/DGND 管脚是否按器件手册建议连接。
- 是否存在跨分割布线却没有就近回流通路。
- 屏蔽地、机壳地、保护地和信号地关系是否定义清楚。

## 常见失误

- 只因为看到 AGND/DGND 就强行整板分成两块地。
- 数字时钟线从模拟参考源下方穿过。
- 模拟输入回流绕远路回到同一点地，形成巨大环路。
