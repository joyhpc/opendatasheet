# 分流电阻电流采样布局

## 适用场景

适用于电源监控、马达驱动、电池管理、eFuse、DC/DC 电流检测和精密采样。

## 关键规则

- 采样精度首先取决于 Kelvin 连接，其次才是放大器精度。
- 大电流路径和采样引线必须分离，不要让铜皮压降混进采样结果。
- 低边采样简单，高边采样系统兼容性更好；选择时要看共模范围和故障需求。
- 分流电阻功耗、温漂和脉冲能力必须与系统最大工况匹配。

## 评审清单

- 采样放大器输入是否直接 Kelvin 接到 shunt 两端。
- 大电流进出方向是否明确，避免采样点跨越大面积铜皮。
- 放大器共模输入范围是否覆盖最坏输入和瞬态。
- RC 滤波是否影响环路稳定或响应速度。
- shunt 的热漂移是否会破坏校准精度。

## 常见失误

- 采样引线从大电流铜皮远端引出，导致读到的是板内压降。
- 只看额定电阻值，不看温漂和脉冲过载。
- 前端滤波电阻过大，导致采样放大器输入偏置误差明显放大。
